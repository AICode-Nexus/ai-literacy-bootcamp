# AI 素养训练营

> 面向产品、运营、市场、HR 等非研发岗位的 AI 能力提升课程

[![VitePress](https://img.shields.io/badge/VitePress-1.6-646cff)](https://vitepress.dev/)
[![License: CC BY-NC-SA 4.0](https://img.shields.io/badge/License-CC%20BY--NC--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-sa/4.0/)

## 课程定位

**不是教"如何写代码"，而是教"如何用 AI 把工作做得更好、更快、更聪明"。**

- **受众**：产品经理、运营、市场、HR、行政、销售等非研发岗位
- **总课时**：8 节主课 + 实战专题，约 19 小时
- **形式**：场景驱动 + 动手实操 + 案例分析
- **特色**：零代码门槛，聚焦真实工作场景

## 在线访问

课程部署为 VitePress 站点：

```bash
# 本地开发
npm install
npx vitepress dev docs

# 构建
npx vitepress build docs
```

## 核心理念

1. **场景优先** — 每个知识点都对应真实工作场景
2. **动手为主** — 60% 时间用于实操练习
3. **即学即用** — 当天学完，明天就能用在工作中
4. **持续迭代** — AI 工具快速演进，培养自学能力
